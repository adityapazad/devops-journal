# UTILITY COMMANDS

- `history` OR `history | grep <command>` - display previously used commands
- `<command> --help` OR `<command> --help | more` - check syntax and option available for a command
- `man <command>` - read more info about command
- `which <command>` - check which executable (file) is being used for a command
- `bs` - calculator
- `cal` - calendar
- `uptime` - check how long server has been running on linux
- `script` - record your activity on terminal in a file
- `alias <a="ls -al">` - create shortcut of a long command in linux (NOT PERMANENT)
