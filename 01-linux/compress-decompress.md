# COMPRESS AND DECOMPRESS FILES

- `gzip -k <file>` - compresses a file (-k = keep the original)
- `gzip -d <file>` OR `gunzip <file>` - decompress a file (-d = decompress)
- `tar czf <NewFile>.tar.gz <folder>` - compress a folder (czf = compress zip folder)
- `tar -xzf <file>` - decompress a folder's file
- `zip <NewFile>.zip <file1> <file2>` - compress multiple files in one zip file
- `unzip -l <file>` - list files in zipped file
- `unzip <file>` - unzip file
